# บทที่ 3 — มุมมองระดับสูงของฟังก์ชันและการเชื่อมต่อคอมพิวเตอร์

## 3.1 องค์ประกอบของคอมพิวเตอร์ (Computer Components)

ระบบคอมพิวเตอร์ประกอบด้วยองค์ประกอบหลัก 4 ส่วน ได้แก่

| องค์ประกอบ | คำอธิบาย |
|---|---|
| **หน่วยประมวลผลกลาง (Processor / CPU)** | ควบคุมการทำงานของคอมพิวเตอร์และดำเนินการประมวลผลข้อมูล |
| **หน่วยความจำหลัก (Main Memory)** | เก็บข้อมูลและโปรแกรมที่กำลังใช้งาน (Volatile — ข้อมูลสูญหายเมื่อปิดเครื่อง) |
| **โมดูล I/O (I/O Module)** | รับส่งข้อมูลระหว่างคอมพิวเตอร์กับอุปกรณ์ภายนอก เช่น ดิสก์ เทอร์มินัล |
| **บัสระบบ (System Bus)** | โครงสร้างการสื่อสารที่เชื่อมโยงระหว่าง Processor, Memory และ I/O |

### ความสัมพันธ์ระหว่างองค์ประกอบ

- **CPU** อ่านคำสั่ง (Instruction) จาก **Main Memory** แล้วดำเนินการตามคำสั่งนั้น
- ข้อมูลสามารถแลกเปลี่ยนกันผ่าน **System Bus** ซึ่งประกอบด้วยสายสัญญาณหลายประเภท
- **I/O Module** ทำหน้าที่เป็นตัวกลางระหว่างระบบคอมพิวเตอร์กับโลกภายนอก

# บทที่ 3 — มุมมองระดับสูงของฟังก์ชันและการเชื่อมต่อคอมพิวเตอร์

## 3.7 คำศัพท์สำคัญ คำถามทบทวน และโจทย์ (Key Terms, Review Questions, and Problems)

---

### คำศัพท์สำคัญ (Key Terms)

| คำศัพท์ (อังกฤษ) | คำแปล / ความหมาย |
|---|---|
| **Address bus** | บัสที่ส่งสัญญาณระบุตำแหน่ง Memory หรือ I/O |
| **Bus** | เส้นทางการสื่อสารร่วมที่เชื่อมต่ออุปกรณ์หลายชิ้น |
| **Bus arbitration** | กระบวนการตัดสินว่าอุปกรณ์ใดจะได้ครอบครองบัส |
| **Bus width** | ความกว้างของ Data Bus วัดเป็นจำนวนบิต |
| **Control bus** | บัสที่ส่งสัญญาณควบคุม เช่น Read, Write, Interrupt |
| **Data bus** | บัสที่ส่งข้อมูลระหว่างโมดูล |
| **DMA (Direct Memory Access)** | การถ่ายโอนข้อมูลโดยตรงระหว่าง I/O และ Memory โดยไม่ผ่าน CPU |
| **Instruction cycle** | วงจรการดึงและดำเนินการคำสั่งหนึ่งคำสั่ง |
| **Interrupt** | สัญญาณที่บอกให้ CPU หยุดชั่วคราวเพื่อให้บริการงานเร่งด่วน |
| **Interrupt handler** | โปรแกรมที่ทำงานเมื่อเกิด Interrupt |
| **I/O module** | โมดูลที่ควบคุมการรับส่งข้อมูลกับอุปกรณ์ภายนอก |
| **PCIe (PCI Express)** | มาตรฐานการเชื่อมต่อแบบ Point-to-Point ความเร็วสูง |
| **Point-to-Point interconnect** | การเชื่อมต่อโดยตรงระหว่างอุปกรณ์สองชิ้น |
| **QPI (QuickPath Interconnect)** | เทคโนโลยีการเชื่อมต่อแบบ Point-to-Point ของ Intel |
| **System bus** | บัสหลักที่เชื่อมต่อ CPU, Memory และ I/O |

---

### คำถามทบทวน (Review Questions)

1. ส่วนประกอบหลักสี่ส่วนของคอมพิวเตอร์มีอะไรบ้าง และแต่ละส่วนทำหน้าที่อะไร?

2. จงอธิบายขั้นตอนของวงจรคำสั่ง (Instruction Cycle) ตั้งแต่ต้นจนจบ

3. Interrupt คืออะไร และทำไมจึงมีประโยชน์ต่อประสิทธิภาพของระบบ?

4. System Bus ประกอบด้วยบัสประเภทใดบ้าง อธิบายหน้าที่ของแต่ละประเภท

5. ปัญหาของการใช้ Shared Bus คืออะไร และแก้ไขได้อย่างไร?

6. การเชื่อมต่อแบบ Point-to-Point แตกต่างจาก Shared Bus อย่างไร?

7. QPI มีกี่ชั้น และแต่ละชั้นทำหน้าที่อะไร?

8. PCIe Transaction Layer รองรับการดำเนินการประเภทใดบ้าง?

9. PCIe Data Link Layer มีกลไกอะไรในการรับประกันความน่าเชื่อถือของข้อมูล?

10. จงเปรียบเทียบความเร็วของ PCIe แต่ละเวอร์ชัน

---

### แนวคิดสำคัญที่ควรจำ

> **วงจรคำสั่งพื้นฐาน:** Fetch → Decode → Execute → Store

> **Interrupt ช่วยเพิ่มประสิทธิภาพ** โดยให้ CPU ทำงานอื่นได้ระหว่างที่รอ I/O

> **ระบบบัสลำดับชั้น** ช่วยลดคอขวดโดยแบ่งการจราจรข้อมูลออกเป็นหลายระดับ

> **PCIe และ QPI** เป็นตัวแทนของแนวทาง Point-to-Point ที่ให้แบนด์วิดท์สูงและความหน่วงต่ำ
